라이브 방송 시작

    날방TV 방송 시작 방법을 순서대로 알아봅시다!

    • 1. 송출앱 실행

      날방TV 방송을 촬영하고 송출하기 위해서는 [날방TV 방송 앱]을 설치 및 실행해야 합니다.

    • 2. 방송 불러오기

      ⓐ 방송 앱을 통해 [날방TV 운영자 웹 페이지]에서 생성한 방송을 확인할 수 있습니다.

      ⓑ 생성된 방송이 여러 개일 경우 예약 시간이 가장 빠른 방송을 불러옵니다.

    • 3. 생방송 전 리허설

      ⓐ 실제 방송을 하기 전에 리허설 방송을 통해 방송장비가 연결된 인터넷의 송출 상태를 점검하고 조경, 배경 등 방송환경을 미리 확인하실 수 있습니다.

      ⓑ 방송 앱에서 [리허설 방송]을 선택 후 촬영을 시작합니다.

      ⓒ [날방TV 운영자 웹 페이지/방송 관리 화면]에서 리허설 방송을 모니터링하고 품질과 상태 점검을 진행합니다.

    • 4. 생방송 시작

      [날방TV 운영자 웹 페이지/방송 관리 화면]에서 방송 시작 버튼을 클릭하여 활성화하면 방송 시작 준비가 완료됩니다.

      미리 설정한 방송 시작 시간 30분 전에 방송을 시작하실 수 있습니다.

    • 5. 라이브 송출

      ⓐ 방송 앱에서 송출 시작을 통해 본 방송 촬영 및 영상 송출을 시작합니다.

      ⓑ 방송 앱에서 촬영 시작 이후 [날방TV 운영자 웹 페이지/방송 관리 화면]을 통해 날방TV 생방송 진행을 관리합니다.

      ⓒ [날방TV 운영자 웹 페이지/방송 관리 화면]에 생성된 [날방TV 라이브 URL 페이지]를 열어 진행 중인 방송 화면을 시청할 수 있습니다.

  • 날방TV 방송 앱 설치 및 실행

    • 날방TV 방송앱은 iOS와 Android 기기를 지원합니다.

  • 방송 송출 화면 기능 설명

    1. ⓐ 방송 송출 방식

      방송 송출 방식은 리허설/라이브 두 가지 방식이 제공됩니다.

    2. ⓑ 카메라 전환 기능

      카메라를 앞/뒤로 전환 선택할 수 있는 기능이 제공됩니다.

    3. ⓒ 방송 시작 버튼

      • 시작 버튼을 누르시면 라이브 송출이 시작됩니다

      • 라이브를 송출하실 때에는 반드시 날방TV 운영자 웹 페이지에서 방송을 시작하신 후 송출하여 주세요.

      • 정지 버튼을 누르면 송출이 중단되며, 송출이 중단이 되어도 방송이 정지되지 않습니다.

      • 방송 정지는 앱이 아닌 날방TV 운영자 웹 페이지에서 종료하여야 합니다.

    4. ⓓ 좌우 반전 기능

      송출 화면을 좌/우 반전 시키는 기능이 제공됩니다.

    5. ⓔ 음소거 기능

      송출 중 음소거 버튼을 누르시면 방송 음소거 또는 음소거를 해제할 수 있습니다.

  • 리허설 방송 송출하기

    • 생방송 전에 리허설 기능을 통해 방송 환경, 네트워크 및 장비 등의 상태를 미리 테스트할 수 있습니다.

    1. 1. 방송 불러오기

    2. 2. 리허설 준비

      ⓐ 리허설 탭을 선택해 리허설을 활성화합니다.

    3. 3. 리허설 방송 시작

      ⓑ 리허설 시작하기 버튼을 눌러 리허설 방송 촬영 화면으로 전환합니다.

    1. 4. 리허설 방송 링크

      날방TV 운영자 웹 페이지에서 ⓒ리허설 링크를 복사하여 리허설 방송 상태를 확인합니다.바로가기

  • 리허설 방송 송출하기

    • 생방송 전에 리허설 기능을 통해 방송 환경, 네트워크 및 장비 등의 상태를 미리 테스트할 수 있습니다.
    1. 1. 방송 불러오기

      날방TV 송출 앱을 실행시키고 방송을 불러옵니다.

      ⓐ 라이브 탭을 선택해 라이브를 활성화합니다.

    3. 3. 리허설 방송 시작

      ⓑ 라이브 시작하기 버튼을 눌러 라이브 방송 촬영 화면으로 전환합니다.

    4. 4. 방송 촬영 시작

      방송 촬영 화면에서 라이브 방송을 시작합니다.

    1. 5. 생방송 송출

      날방TV 운영자 웹 페이지 화면에서 ⓒ방송 시작을 버튼을 눌러 생방송 송출을 시작합니다.

    2. 6. 방송 확인

      날방TV 운영자 웹 페이지 화면에서 라이브 방송 상태를 확인하고 ⓓ라이브 링크를 통해 실제 라이브 방송을 확인합니다.

    • PC 송출 안내

    • 날방TV는 앱을 사용하지 않더라도 방송 송출을 할 수 있도록 RTMP를 통한 방송 송출을 지원합니다.
    • RTMP 기반으로 송출이 가능한 PC/Mac 용 소프트웨어가 필요합니다. (프리즘, OBS 등)

    • 1. 방송 생성

    • 2. 날방TV 운영자 웹 페이지에서 RTMP 정보 확인

      유료 서비스에 한하여 지원

    • 3. OBS에서 RTMP/Stream Key 세팅

    • 4. 날방TV 운영자 웹 페이지에서 방송 시작

    • 5. OBS에서 라이브 송출 시작

    OBS를 이용해 날방TV 동시 송출하기

    1. 1. [날방TV 운영자 웹 페이지]에서 방송을 생성하면 개별 방송에는 RTMP, Stream Key가 제공됩니다.

    2. 2. 라이브 방송 전에 날방TV 담당자에게 RTMP URL과 Stream Key 발급 요청을 합니다.

    3. 3. 제공된 RTMP URL과 Stream Key를 OBS의 설정 > 방송란에 입력하여 송출을 준비합니다.

    1. 4. OBS 안의 ⓐ설정 버튼을 클릭하면 값을 입력할 수 있는 설정 팝업창이 노출됩니다.

    1. 5. OBS 설정 팝업창 왼쪽 메뉴 중 방송 버튼을 클릭하면 ⓑRTMP URL과 Stream Key를 입력할 수 있는 페이지가 노출됩니다.

    2. 6. ⓒ서버 항목에 발급받은 RTMP URL을 입력합니다.

    3. 7. ⓓ스트림 키 항목에 Stream Key를 입력합니다.

      • OBS에서 동시 송출할 다른 방송 채널과 각종 시스템 설정을 진행합니다.
      • 실제 송출을 위한 출력 옵션 값, 비디오 설정값 등은 실제 송출 환경에 맞게 설정하셔야 합니다. 자세한 설정법은 OBS 자체 도움말을 참고해 주세요.

    4. 8. OBS 설정이 완료되면 라이브 방송을 시작하기 위해 ⓔ방송 시작 버튼을 눌러 방송 중인 상태로 변경합니다.

    1. 9. OBS에서 ⓕ방송 시작 버튼을 눌러 라이브 영상을 송출합니다.
